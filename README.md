
# Install Composer Dependencies

```
composer install
```

# Set Up Environment Variables

```
cp .env.example .env
```

# Generate an Application Key

```
php artisan key:generate
```

# Set Up the Database

```
php artisan migrate --seed
```

# Serve the Application

```
php artisan serve
```
